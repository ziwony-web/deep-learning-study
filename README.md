The study covered core concepts in deep learning, including neural networks, convolutional neural networks, recurrent neural networks, attention mechanisms, and Transformers.

- Convolutional Neural Networks
- Pooling Layers
- Recurrent Neural Networks
- Attention Mechanisms
- Transformers
